FAQ

Totally fair to ask.

HermesWork is built for independent builders who want a real operating system, not a wireframe or a SaaS demo.

Is this real data or demo data?
Real data only. The dashboard stays empty until you create real invoices or Stripe test webhooks confirm payments. Only Stripe runs in test mode — no other demo layer.
What does “Stripe test mode” mean?
Full Stripe integration using test keys — so you can run the entire payment flow, see webhooks fire, and confirm payments without moving real money.
What is the x402 rail?
A crypto-native payment-proof route using USDC on Base Sepolia. Payment proof is verified and recorded the same way as Stripe — one record, two rails.
How do I deploy it?
Render: backend as a web service, frontend as a static site. No credit card required, free SSL, auto deploys from GitHub every push.
What are ERC-8004 work records?
A portable credential standard for freelancers. Every confirmed payment can mint a record — verifiable, on-chain proof of paid work that you own and can carry anywhere.
