FAQs

Totally fair to ask.

How does the waitlist work? +

HermesWork is in closed beta. Join via GitHub, clone the repo, and follow the deploy guide. Full waitlist opens at the NousResearch Hackathon deadline (June 30, 2026).

Is there a free plan? +

Yes. Oracle Cloud gives you 4 vCPUs and 24GB RAM permanently free. Hermes, NemoClaw, OpenRouter, and Stripe test mode — all zero cost.

How does ERC-8004 reputation work? +

After every confirmed payment (Stripe or x402), HermesWork mints an ERC-8004 credential on Base Sepolia. It contains your job type, amount, payment proof, and timestamp — portable across any compatible platform.

What payment methods are supported? +

Stripe (cards, bank transfers in USD/EUR/GBP/INR) and x402 Protocol (USDC on Base Network, sub-second settlement, <$0.01 gas). Both rails fire automatically.

Can I invite my team? +

Multi-agent support via Antigravity 2.0 is in development. Each team member gets their own Hermes agent sharing a common MEMORY.md.

Is my data secure? +

All agent operations run inside NemoClaw, NVIDIA's kernel-level sandbox. Client data never leaves the secure perimeter. API keys are timing-safe compared and stored only in your browser.

What if I need help? +

Join the NousResearch Discord, file a GitHub issue, or message the Telegram bot. The Hermes community is active and growing fast.
