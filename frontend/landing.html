FAQ

Questions, answered.

Built for builders.

HermesWork keeps real invoices, payments, and proof in one place so you spend time on the work, not the admin.

Is this real data or a demo?

The dashboard is real-data-first. It stays empty until you create real records or Stripe test webhooks confirm payments. Only Stripe runs in test mode.

What does Stripe test mode mean?

Payments use real Stripe integration with test keys, so you can run the full flow safely without moving real money.

What is the x402 rail?

A crypto-native payment-proof route using USDC on Base Sepolia, with a verifiable proof flow that updates the same ledger.

How do I deploy it?

Render: backend as a web service, frontend as a static site. No credit card, free SSL, auto deploys from GitHub.
