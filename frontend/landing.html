FAQ

Totally fair to ask.

No closed beta, no waitlist — open the dashboard and start.

Is this real data or a demo?
Real data only. The dashboard starts empty and fills as you create invoices, confirm payments, and receive webhooks. No seed data, no sample rows to clean up.
What does Stripe test mode mean?
Full Stripe integration using test-mode keys. You run the complete payment flow — hosted page, webhook, confirmation — without any real money involved. Swap to live keys when ready.
What is the x402 payment rail?
A crypto-native payment-proof route. Your client submits a USDC payment on Base Sepolia, the backend verifies the on-chain proof, and the invoice is marked confirmed — same flow as Stripe.
How do I deploy for free?
Render: backend as a Web Service (root dir: backend), frontend as a Static Site. No credit card required. Free SSL. Auto-deploys from GitHub on every push.
What are ERC-8004 work records?
A portable credential for freelancers. Every confirmed payment generates a signed on-chain record that proves you did the work and got paid — no middleman, no platform lock-in.
