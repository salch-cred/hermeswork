## ⚡ Install as a Hermes Agent Skill

```bash
# One command — HermesWork becomes a native Hermes skill
mkdir -p ~/.hermes/skills/business/hermeswork
curl -o ~/.hermes/skills/business/hermeswork/SKILL.md \
  https://raw.githubusercontent.com/salch-cred/hermeswork/main/skills/hermeswork/SKILL.md
```

Then in Hermes Agent:
```
/hermeswork
"Create an invoice for Acme Corp for $2500 due next week"
"Find me 3 freelance jobs to pitch today"
"What's my cash runway?"
"Generate a contract for the new client"
"Run the monthly board report"
```

## 🚀 Key Endpoints

```bash
# System
GET  /health                    # Status, agent count, all integrations
GET  /agents                    # All 27 agents + papers
GET  /mcp/manifest              # 48 MCP tools
GET  /.well-known/agent.json    # A2A agent card
GET  /.well-known/mpp.json      # Merchant payment profile

# Business Operations
GET  /kpis                      # Live KPIs
POST /invoices                  # Create invoice + Stripe link
PATCH /invoices/:id/pay         # Mark paid
POST /proposals                 # Track proposal
PATCH /proposals/:id            # Update status (auto-triggers onboarding on win)

# AI Agents
POST /ai/forecast               # ARIMA revenue forecast
POST /ai/coach                  # Win rate coaching
POST /ai/contract               # Generate contract
POST /ai/board-report           # Monthly board report
POST /ai/job-scout              # 🆕 Autonomous job discovery
POST /ai/runway                 # 🆕 Cash flow runway prediction

# Automations
POST /automations/collect       # Run collection agent now
GET  /automations/status        # All automation status

# Channels
GET  /bot/setup                 # Register Telegram webhook
GET  /whatsapp/status           # WhatsApp status
```

## 🛠 Setup

```bash
git clone https://github.com/salch-cred/hermeswork
cd hermeswork/backend
npm install
cp .env.example .env
# Fill in env vars below
npm start
```

### Required
```env
HERMESWORK_API_KEY=your_secret_key
NVIDIA_NIM_API_KEY=nvapi-...        # NVIDIA NIM for Hermes 3
STRIPE_SECRET_KEY=sk_test_...
PUBLIC_BASE_URL=https://your-backend.onrender.com
```

### Telegram Bot
```env
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHAT_ID=...
# After deploy: GET /bot/setup to register webhook
```

### WhatsApp Bot (Twilio)
```env
TWILIO_ACCOUNT_SID=...
TWILIO_AUTH_TOKEN=...
TWILIO_WHATSAPP_FROM=whatsapp:+14155238886
# Set webhook in Twilio Console → Messaging → WhatsApp Sandbox
```

### Blockchain (optional)
```env
PRIVATE_KEY=0x...          # Base Sepolia wallet
ERC8004_REGISTRY=0x...     # Deployed ERC-8004 contract  
PAYMENT_ADDRESS=0x...      # USDC receive address
```
